Como chamr um CSS:

O CSS e responsavel pela formataçao do seu web site, colca cor, estilo, tamanho e loalizaçao dos elementos
html.
existem 3 formas de chamar uma pagina CSS para a sua pagina html:

     *maneira externa: Criar uma pagina CSS que sera linkada a paginas html
     *adicionar dentro da tag <head>: Podemos adicionar as formataçoes em CSS O chamando dentro da tag
     head, assim apenas aquela pagina sera formatada!
     *na tag <HTML>: 

     para likarmos uma pagina css com um pagina html usaremos dentro de <head> a tag <link> e colocaremos
     no atributo href= o link para nossa pagina
     
     para formatrmnos uma tag primeiro a puxamos ex:
     div{}, tudo que ficar dentro da de {} sera parte da formataçao de div, tudo que colocarmos sera uma 
     propriedade:

     color:

     e tudo apos : sera o valor dela:

     color:grenn

     como em C++ devemos encerrar com ; a cada comando escrito:

     color:grenn;
